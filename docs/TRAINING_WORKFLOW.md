## 中文

### 1. 文档目的

说明如何基于真实样本持续训练和校准安全日志分析主线。

### 2. 训练输入

训练应至少具备原始样本、目标输出、分类边界和报告结构要求。

### 3. 标准流程

标准流程包括样本进入、分类校准、报告校准和学习沉淀。

### 4. 当前重点

当前重点训练主题为 Host baseline、JumpServer 多源审计、Whitebox AppSec，以及 EASM 单样本与多样本综合分析。

### 5. 注意事项

必须验证表头、行数、聚合、下载与页面一致性，不能只验证解析成功。
